# Harness Design

How an agent perceives its environment and acts on it — the interface layer between the model and the world.

**In scope:** action/tool definitions and their schemas, the shape of observations and tool results, how errors and failures are surfaced back to the model, retry and recovery affordances, action-space size and ergonomics, result formatting that raises completion rates.

**Out of scope:** the wording of prompts (→ [prompting/](../prompting/)), how multiple agents are wired together (→ [orchestration/](../orchestration/)).
