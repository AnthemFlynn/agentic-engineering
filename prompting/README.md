# Prompting

Prompt and context engineering — what goes into the context window and how it's arranged.

**In scope:** system-prompt design, instruction phrasing, context-window management, compaction and summarization, few-shot construction, ordering and salience, steering and refusal handling.

**Out of scope:** the tools and observations the prompt references (→ [harness-design/](../harness-design/)), how outputs are scored (→ [evals/](../evals/)).
