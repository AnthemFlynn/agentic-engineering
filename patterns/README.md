# Patterns

Cross-cutting reusable patterns that don't sit cleanly under a single topic.

**In scope:** techniques that span areas — e.g. content-hash caching of expensive work, iterative/progressive retrieval, regex-vs-LLM decision rules for structured text, search-first workflows.

**Out of scope:** anything that clearly belongs to one area — file it there instead. This is the home for the genuinely orthogonal. If several entries here share a theme, promote them to their own area.
