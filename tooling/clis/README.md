# CLIs

Standalone command-line tools that agents drive or depend on — not full harnesses, but the utilities around them.

**In scope:** invocation patterns, flags, and gotchas for tools agents commonly call (e.g. `gh`, `rg`, `jq`, `git`), and purpose-built CLI helpers for agentic workflows.

**Out of scope:** the agent harnesses themselves (→ [../harnesses/](../harnesses/)).
