# Skills

Agent skills as a **cross-harness concept** — packaged, reusable units of specialized capability.

**In scope:** skill ideas and authoring principles that aren't tied to one harness's file format or invocation mechanism.

**Out of scope:** anything Claude-Code-specific — `SKILL.md` structure, frontmatter, `user-invocable`/`argument-hint`, discovery, and concrete skill recipes live under [../harnesses/claude-code/](../harnesses/claude-code/). What belongs here is cross-harness: portable skill patterns and references to published skills that work unmodified across runtimes.
