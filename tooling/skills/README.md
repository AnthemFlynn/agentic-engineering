# Skills

Agent skills — packaged, reusable units of specialized capability and knowledge.

**In scope:** skill authoring and structure (SKILL.md, frontmatter, triggers), discovery and routing, distribution and versioning, and cross-harness skill patterns.

**Out of scope:** harness-specific skill plumbing, which goes under the relevant harness (e.g. [../harnesses/claude-code/](../harnesses/claude-code/)).
